✅ TAREA 4: WHATSAPP - IGNORAR CONTACTOS CONOCIDOS + WELCOME NEW USERS

Problema:
Bot spammea a todos con welcome, incluso contactos frecuentes

Solucion Implementada:
- Creada tabla 'knownContacts' en schema
  (platformId, platform, targetPlatformId, firstContactAt)
- Logica en primer mensaje por usuario:
  * Si es NUEVO CONTACTO -> INSERT en DB + envia welcome
  * Si es CONTACTO CONOCIDO -> ignora (sin mensaje)

Codigo implementado:
const existing = await db.select().from(knownContacts)
  .where(and(
    eq(knownContacts.platformId, userId),
    eq(knownContacts.platform, 'whatsapp')
  ))

if (!existing) {
  // NUEVO CONTACTO: INSERT + welcome
  await db.insert(knownContacts).values({...})
  await sock.sendMessage(jid, {text: "¡Hola! Bienvenido..."})
}
